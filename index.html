Projects / Case Studies

Automated Regression Framework — Orders over FIX

Problem

Manual regression took 2 weeks which was time/resource intensive.

Solution

Designed a modular/data driven Python test framework using Pytest, integrated with CI/CD, and built API and DB validation checks.

Outcome

Regression time reduced to <1 day; defects caught earlier resulting in faster release sign-off.

Manual Orders Intervention

Problem

Most test scenarios were difficult/impossible to trigger manually.

Solution

Collaborated with dev team to build a test simulator written in Python to allow the testing of vanilla and edge case scenarios.

Outcome

Test cycle timeframes significantly reduced and also limited the need for external team involvement.

Automated Controls Testing Framework

Problem

Gaps in front-to-back trade controls (limit checks, validation rules, and throttling) led to undetected breaches and potential regulatory exposure.

Solution

Defined and executed a comprehensive QA strategy to test key controls across trade capture, validation, and reporting. This included automation of limit checks, negative test scenarios, and workflow testing.

Outcome

The strengthened control testing framework identified multiple control weaknesses before go-live, improved compliance confidence, and ensured that regulatory and risk controls operated reliably under production conditions.

Automated Task Manager Framework

Problem

Manual testing of the task manager web application was time-consuming and error-prone, especially for verifying dynamic features like task creation, due dates, and user interactions across multiple table rows.

Solution

Developed a Pytest BDD + Playwright automation framework to validate the task manager end-to-end. Feature scenarios were written in Gherkin to cover key workflows and Pytest BDD structured tests were developed for maintainability and readability.

Outcome

Achieved reliable automated verification of task creation, updates, and calendar interactions that ensures the task manager behaves correctly after each update.

Trader UAT Coordination — BARX FX Trading Platform

Problem

Traders experienced repeated environment issues and unclear test scripts.

Solution

Created detailed UAT packs and environment checklists. Coordinated daily triage calls.

Outcome

Smooth UAT cycles and adhesion to project timelines.
